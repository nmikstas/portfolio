Digital Down-Converter

Digital Down-Converter Overview

The DDC was to be designed with the following parameters:
1. The decimator input sample rate is 30.72MHz
2. The input signal is complex
3. The FPGA clock frequency is 245.76MHz
4. The peak filter sidelobe is to be -60dB with respect to the passband
